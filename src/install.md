# How to install Jetpy

## Windows

1. Donwload [Jetpy](https://github.com/vivavy/jetpy")
2. Unpack it
3. Copy `src\jetpy\winjetpy.py` to `C:\Windows\System32\jetpy.py`
4. Copy `src\jetpy` to `C:\Jetpy`
5. Done!

## Linux

1. Donwload [Jetpy](https://github.com/vivavy/jetpy")
2. Unpack it
3. go to `src`
4. run `make install`
5. Done!

## MacOS

1. Donwload [Jetpy](https://github.com/vivavy/jetpy")
2. Unpack it
3. go to `src`
4. run `make install`
5. Done!
